Tools I use and how I use them:

I use WSL (Windows Subsystem for Linux) as my default terminal. I prefer Linux commands over PowerShell, simply because they are about a dozen times more intuitive. Being able to install and use any package I want really helps the development process for me as well.

Windows Terminal + WSL running Debian

I use Microsoft's Visual Studio Code as my default code editor. The tons of really useful extensions, Git integration and the built-in terminal greatly improve my coding efficiency.

Visual Studio Code + WSL Remote

I use a VPS server for tasks such as webhosting. As a matter of fact, this very website is hosted on my VPS. I do this via the XAMPP package, which is kind enough to give me an FTP server and database as well. This VPS is also used for some of my personal experiments.

It costs 6 EUR a month, has a (virtual) 4c4t AMD EPYC CPU , 8GBs of RAM and a 250GB SATA SSD.

Debian VPS server running XAMPP

My laptop runs a dual boot installation. One half is Windows 10, the other Manjaro Linux. Manjaro runs on the Arch kernel and is almost identical to it. It comes with an easy installer and stability fixes. I use the i3-gaps window manager because I consider it to be the most efficient way to use a computer.

During my study at Deltion I will mostly be using Windows.

Manjaro Linux running on my laptop

I have my own cloud server running on a Pi 4 (8GB RAM). The server hosts CIFS shares that I can mount on both Windows, Linux and mobile. The CIFS shares are mostly used to stream media and store Deltion assignments. I use WireGuard as a VPN server to connect to my cloud from external networks.

This Pi also runs Docker, with Portainer acting as the interface. I run a qBittorrent container which is hooked up to an ExpressVPN container.

Raspberry Pi 4 running OpenMediaVault

I use Firefox as my main development browser because I am a self-respecting webdeveloper. Firefox's dev-tools make using CSS Flex a ton easier.

My secondary browser is Brave. Brave is a browser running on Google's Chromium. I use both of these webbrowsers to test and develop my webapplications.
