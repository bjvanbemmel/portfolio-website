Hulpmiddelen die ik gebruik:

Ik gebruik WSL (Windows Subsystem for Linux) als mijn standaard terminal. Linux's commando's werken voor mij vele malen beter dan PowerShell's. Moeiteloos packages kunnen downloaden en gebruiken versnelt het programmeer proces ook enorm voor mij.

Windows Terminal + WSL running Debian

Ik gebruik Microsoft's Visual Studio Code als mijn standaard code editor. Code komt met handige ingebouwde functies, groote hoeveelheid plugins en de vlekkeloze git integratie verbeterd mijn leven enorm.

Visual Studio Code + WSL Remote

Ik heb een VPS server voor taken zoals webhosting. Deze website draait bijvoorbeeld ook op mijn VPS. De VPS server draait XAMPP. XAMPP geeft mij ook een FTP server en database. Ook word deze VPS gebruikt voor persoonlijke experimenten.

Hij kost me 6 EUR per maand, heeft een (virtuele) 4c4t AMD EPYC CPU , 8GBs RAM en een 250GB SATA SSD.

Debian VPS server running XAMPP

Mijn laptop draait een dual boot installatie. Één helft is Windows 10, de ander Manjaro Linux. Manjaro draait op Arch. Het komt met een makkelijkere installatie dan Arch en is een stuk stabieler. Ik gebruik de i3-gaps window manager omdat ik dit de meest efficiënte manier vind om een computer te gebruiken.

Op Deltion ga ik voornamelijk Windows 10 gebruiken.

Manjaro Linux running on my laptop

Ik heb mijn eigen cloud server draaien op een Raspberry Pi 4 (8GB). Deze server hosts CIFS shares die ik kan mounten op Windows, Linux en mobiel. De CIFS shares worden grotendeels gebruikt om media te streamen en om mijn Deltion opdrachten te archiveren. Ik gebruik WireGuard als VPN server om te verbinden met mijn Pi vanaf buiten mijn thuis-netwerk.

Ook draait mijn Pi Docker, met Portainer als de interface. Mijn Docker installatie draait een qBittorrent container die gekoppeld is aan een ExpressVPN container.

Raspberry Pi 4 running OpenMediaVault

Ik gebruik Firefox als development browser omdat ik een zelf-respecterende webdeveloper ben. Firefox's dev-tools maakt het ontwikkelen met CSS Flex vele malen makkelijker.

Firefox as main development browser
